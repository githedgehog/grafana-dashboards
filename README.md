## Instructions on updating

0. Select import when creating a dashboard
0. Enter the number of the desired dashboard 
0. Make the edits / updates to the dashboard
0. Ensure to add a commit message for the changes
0. Exit the edit mode of the dashboard
0. If the look and feel of the dashboard has changed export the dashboard as an
   image to update the screen shot.
0. Click the `export` button in the top left of the dashboard 
0. Ensure the "Export for sharing externally" slider is on
0. Replace the existing JSON with the newly exported in this repository
0. Upload the new json to the webpage on `grafana.com`.
